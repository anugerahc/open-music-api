# Open Music API

### How To Use

> Clone Repository
``` bash
$ git clone https://github.com/MEY-Mental-Education-Yes/MEY-APP-BACKEND.git
```
>Go to Directory
``` bash
$ cd open-music-api
```
>Install Dependencies
``` bash
$ npm install
```
>Migrate Database using Node PG Migrate
``` bash
$ npm run migrate up
```
>Migrate Database using Node PG Migrate
